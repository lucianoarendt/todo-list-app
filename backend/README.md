# BACKEND

## To Start

Go Install V.16
`https://go.dev/doc/install`

## Dependencies:

- Fiber
 to install:
    `go get github.com/gofiber/fiber`

- Gorm
 to install:
    `go get -u gorm.io/gorm`

- Gorm mysql-driver
 to install:
    `go get -u gorm.io/driver/mysql`

## Database schema
 on workbench or any database URI software, connect and run this cmd:
    `CREATE SCHEMA db;`
